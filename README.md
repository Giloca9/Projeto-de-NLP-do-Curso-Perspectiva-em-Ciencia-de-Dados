# Projeto-de-NLP-do-Curso-Perspectiva-em-Ciencia-de-Dados



Este repositório contém um projeto desenvolvido para a disciplina de **Perspectiva em Ciência de Dados**, com o objetivo de aplicar técnicas de **web scraping**, **processamento de linguagem natural (NLP)** e **análise de dados** em notícias relacionadas ao Palmeiras.



O projeto busca coletar notícias publicadas na web, processar os textos obtidos e realizar análises para entender padrões de conteúdo, frequência de palavras, temas recorrentes e possíveis relações entre as características das notícias e o número de visitas.



* **Objetivo do Projeto:**



O principal objetivo do projeto é construir uma solução completa de ciência de dados envolvendo dados textuais, desde a coleta até a análise final.



A proposta consiste em:



\- coletar notícias sobre o Palmeiras por meio de web scraping;

\- organizar informações como título, data, texto da notícia e número de visitas;

\- aplicar técnicas de NLP para limpeza e transformação dos textos;

\- analisar as palavras mais frequentes após remoção de stopwords;

\- gerar visualizações, como nuvem de palavras;

\- explorar padrões nos textos e nas visitas das notícias;

\- criar uma base estruturada para possíveis aplicações de machine learning.





* **Ferramentas:**



O projeto foi desenvolvido em Python, utilizando bibliotecas voltadas para coleta, manipulação, NLP e visualização de dados.
